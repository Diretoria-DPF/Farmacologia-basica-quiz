# 🧠 Simulador de Farmacologia


## 📌 Sobre o Projeto
Uma plataforma web interativa desenvolvida para estudantes da área da saúde testarem e consolidarem seus conhecimentos em Farmacologia Básica. O projeto foi desenhado sob a ótica do **"Pensamento Provador"**, permitindo que os alunos não apenas decorem gabaritos, mas compreendam a lógica biológica, os mecanismos de ação e o *porquê* de cada resposta fisiológica.

## 🔬 Contexto Acadêmico
O simulador foi criado como uma ferramenta de revisão intensiva e preparatória para curso de Farmácia. O objetivo principal é transformar o estudo passivo e denso de artigos em uma prática ativa, traduzindo conceitos complexos de forma didática, rápida e aplicável ao cenário clínico.

## ⚙️ Funcionalidades Principais
* **Banco de Questões Estratégico:** 240 questões estruturadas, inéditas e revisadas.
* **Modo Estudo 📖:** Feedback imediato após cada seleção, exibindo explicações detalhadas focadas no mecanismo fisiológico para uma aprendizagem ativa.
* **Modo Prova 📝:** Simulação de avaliação real com cronômetro cravado em minutos e tela de resultados apenas no final, exigindo foco.
* **Dashboard Analítico:** Análise visual de acertos por tópico, porcentagem geral de domínio e opção de revisar exclusivamente as questões incorretas.
* **Autonomia e Privacidade Total:** Sistema integrado ao `localStorage` do navegador. O progresso é salvo localmente, permitindo pausar e continuar de onde parou sem envio de dados para servidores externos.

## 📚 Tópicos Abordados
O simulador cobre 12 pilares essenciais da farmacologia:
1. **Farmacodinâmica**
2. **Farmacocinética**
3. **AINEs**
4. **Anti-hipertensivos**
5. **Coagulação**
6. **Dislipidemias** 
7. **Glicocorticoides** 
8. **Diuréticos**
9. **ANTIMICROBINOS**
10. **PSICOFÁRMACOS**
11. **FARMACOLOGIA ENDÓCRINA**
12. **SISTEMA NERVOSO AUTÔNOMO**

## 🛠️ Tecnologias Utilizadas
A aplicação é puramente *Client-Side*, operando de forma leve e rápida em qualquer dispositivo.
* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização responsiva, variáveis nativas e animações fluidas.
* **Vanilla JavaScript (ES6+):** Lógica do simulador, manipulação de DOM, filtragem de arrays e armazenamento local (`localStorage`).

## 🚀 Como Acessar e Utilizar
Acesse o simulador diretamente pelo seu navegador (computador ou mobile), sem necessidade de instalação:

👉 **https://diretoria-dpf.github.io/Farmacologia-basica-quiz/**

**Passo a passo para uso:**
1. Selecione os tópicos que deseja revisar no menu lateral.
2. Escolha entre o **Modo Estudo** ou **Modo Prova**.
3. Clique em "Iniciar Quiz" e acompanhe seu progresso!
